# VISIL — Lens Layer

## Overview
VISIL defines how system outputs are visually and structurally framed.

It does not modify signals or interpret meaning.

It only controls perception.

---

## Core Function

VISIL determines:

- what is visible
- what is hidden
- what is emphasized
- what is de-emphasized

---

## Rules

- VISIL does not alter signal data
- VISIL does not interpret meaning
- VISIL does not execute actions
- VISIL only transforms presentation

---

## Relationship

- Input: Signals from Atrium / Signal Model
- Output: Framed representation for observation
- Downstream: Lumen may interpret VISIL output

---

## Principle

Reality is unchanged.

Only the lens changes.
